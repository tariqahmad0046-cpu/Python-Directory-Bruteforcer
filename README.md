# 🔥 Python Directory Bruteforcer

## 📌 Project Overview

This project is a Python-based directory brute forcing tool used for cybersecurity learning and bug bounty reconnaissance.

It discovers hidden directories and sensitive paths on web servers.

---

## 🎯 Features

- Hidden directory discovery
- Admin panel detection
- API endpoint discovery
- HTTP response analysis
- Beginner-friendly recon tool
